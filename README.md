# The-new-way-to-encrypt
Just a new idea to encrypt the message ,but I think it is safe enough for commom life
系统特点
支持普通加密和超级加密两种模式
超级加密生成外星文般的加密结果，增强隐蔽性
可自定义256字符映射表，支持混合字符类型
使用增强的Unicode编码，确保所有字符正确加密
可配置的加密轮次，提高安全性
新增万能解密模式，采用多重隐藏策略(可惜的是，目前无法实现)
动态模式选择，每次加密使用不同隐藏算法
隐写术保护，防止统计分析攻击
实时字符统计和类型分析
期待你们的破解（狗头）
